@kaviar/grapher

Grapher

Grapher is the fetching layer on top of MongoDB Node Driver, to allow extreme speeds for relational data surpassing SQL in some conditions.

Grapher is designed to work very well with your GraphQL resolvers ensuring you will query the database a minimum amount of time.

The incredible speed boost is possible due to the technology called Hypernova, created by Theodor Diaconu in 2016 for Meteor. Read more about the Hypernova.

Installation

 
npm i -S @kaviar/grapher

Documentation

This provides a learning curve for Grapher and it explains all the features.

Sample

 
import { query, addLinks } from "@kaviar/grapher";

async function test() {
  const Post = await db.createCollection("Post");
  const Comments = await db.createCollection("Comments");

  oneToMany(Comments, Post, {
    linkName: "post",
    inversedLinkName: "comments",
  }); // also available manyToMany and manyToOne

  query(Post, {
      $: {
          filters: {
              isApproved: true
          }
      }
      title: 1,
      comments: {
          name: 1
      }
  })
}

GraphQL

 
import { query } from "@kaviar/grapher";
const Query = {
  // Automatically fetches everything in the minimum amount of queries
  users(_, args, ctx, info) {
    return query.graphql(Post, info).fetch();
  },
};
